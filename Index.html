Quick Review

Most-tested ideas: CPU parts, fetch-decode-execute, MARIE registers, buses, memory, assemblers, and hardwired vs microprogrammed control.

CPU = datapath + control unit ALU does arithmetic/logic Registers are tiny fast storage Bus moves data Clock synchronizes operations Fetch → Decode → Execute Assembler: mnemonic → machine code MARIE = simple teaching architecture Hardwired = faster Microprogrammed = more flexible

Key MARIE Registers

  • AC = Accumulator, holds data/results
  • MAR = Memory Address Register, holds address
  • MBR = Memory Buffer Register, holds memory data
  • PC = Program Counter, next instruction address
  • IR = Instruction Register, current instruction
  • InREG / OutREG = input and output data
